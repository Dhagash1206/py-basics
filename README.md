# py-basics

A collection of Python implementations covering core data structures, algorithms, search strategies, and coding practice problems

---

## 📁 Contents

### 🗂️ Data Structures
| File | Description |
|------|-------------|
| `linked-list.py` | Singly linked list implementation |
| `Doubly-linked-list.py` | Doubly linked list implementation |
| `stack.py` | Stack data structure |
| `queue.py` | Queue data structure |

### 🔍 Search Algorithms
| File | Description |
|------|-------------|
| `A_seach.py` | A* (A-star) search algorithm |
| `greedy_Befs.py` | Greedy Best-First Search (GBFS) |
| `seach_befs.,py` | Best-First Search |
| `uniform_cost_search.py` | Uniform Cost Search (UCS / Dijkstra-style) |
| `DFS.cpp` | Depth-First Search (C++) |

### 🧮 Dynamic Programming & Algorithms
| File | Description |
|------|-------------|
| `fibo_using_dp.py` | Fibonacci sequence using dynamic programming |
| `fibo using memorization` | Fibonacci using memoization |
| `Karatsuba_algorithm.py` | Karatsuba fast multiplication algorithm |
| `LCS_leetcode.py` | Longest Common Subsequence (LeetCode) |
| `rotatearray.py` | Array rotation |
| `OOPS.py` | OOP concepts in Python (classes, inheritance, etc.) |

### 💡 LeetCode / Problem Solving
| File | Description |
|------|-------------|
| `Find-Common-Characters.py` | Find common characters across strings |
| `leetcode-obtain-max-happiness.py` | Maximize happiness sum (greedy) |
| `N-queen` | N-Queens problem using backtracking |

### 🌐 Extras
| File | Description |
|------|-------------|
| `credit-card.py` | Credit card validation logic |
| `weather_api.py` | Weather data fetching via API |

## 🚀 Getting Started

**Prerequisites:** Python 3.x

```bash
# Clone the repository
git clone https://github.com/Dhagash1206/py-basics.git
cd py-basics

# Run any script
python linked-list.py
python A_seach.py


```
